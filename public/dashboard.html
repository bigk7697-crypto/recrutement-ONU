Guide d'Utilisation du Portail Administrateur

1. Gestion des Candidatures

Le tableau de bord affiche en temps réel toutes les candidatures. Vous pouvez filtrer par statut ou rechercher un candidat par son nom ou sa profession. Cliquez sur un candidat pour ouvrir sa fiche complète.

2. Workflow de Sélection

Utilisez le menu déroulant dans la fiche candidat pour faire évoluer le statut :
En attente → Analyse en cours → Entretien → Retenu/Refusé. Chaque changement est historisé automatiquement.

3. Communication

L'onglet "Répondre" permet d'envoyer un message via Email, WhatsApp ou SMS. Notez que si le SMTP n'est pas configuré, le système simulera l'envoi dans les logs.

4. Gestion des Offres & Paramètres

Via la barre latérale, vous pouvez créer de nouvelles offres d'emploi ou modifier vos informations de profil et la configuration officielle de l'organisation.
